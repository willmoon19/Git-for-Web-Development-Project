    37. Walk me through Lambda's git flow. 
        Fork a repo into your github account
        Clone the repo to your computer repo
        cd into the cloned repo
        Create a branch with git checkout -b "name"
        Make adjustments on the project in that branch
        Add any changes you made using git add .
        Commit your changes using git commit -m "detailed message"
        Push your local repo to the remote repo using git push -u origin "branchname"
        Submit a pull request and let another pull your work into the repo
